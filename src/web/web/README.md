# Web Pages

Paginas estaticas minimas para configuracoes da Meta.

Arquivos:

- `privacy.html`
- `terms.html`
- `data-deletion.html`

URLs esperadas quando publicadas:

- `/privacy`
- `/terms`
- `/data-deletion`

Se voce publicar em um host estatico simples, pode mapear:

- `privacy.html` -> `/privacy`
- `terms.html` -> `/terms`
- `data-deletion.html` -> `/data-deletion`
