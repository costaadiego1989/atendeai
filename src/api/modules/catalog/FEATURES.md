# FEATURES - `catalog`

## Estado atual

O modulo gerencia categorias, itens, servicos, produtos, importacao, relatorios, variantes e campos customizados. Ele melhora vendas porque da contexto real para IA, commerce e atendimento humano.

## Features de alto valor

| Prioridade | Feature | Status | Valor para usuario | Criterio de sucesso |
|------------|---------|--------|--------------------|---------------------|
| P0 | Importacao assistida com validacao por linha | parcial | Cliente sobe catalogo rapido sem planilha perfeita. | Importacoes mostram linhas criadas, atualizadas e rejeitadas com motivo. |
| P0 | Variacoes e SKU vendaveis | parcial | Produto com tamanho/cor/opcoes pode ser vendido corretamente no WhatsApp. | Itens com variacao aparecem corretamente no checkout e estoque. |
| P0 | Disponibilidade por filial | recomendado | Cliente evita oferecer item indisponivel na filial do comprador. | Catalogo filtra disponibilidade por branch ativa. |
| P1 | Enriquecimento de descricao por IA | recomendado | Reduz trabalho manual e melhora conversao do catalogo. | Percentual de itens com descricao completa aumenta. |
| P2 | Imagens/CDN e vitrine publica | recomendado | Aumenta confianca e conversao, mas depende de UX/app. | Itens com imagem convertem melhor em fluxos de commerce. |

## Observacao

Catalogo deve ser orientado a venda, nao apenas cadastro. O valor aparece quando IA e checkout usam esses dados.
